# Raycast Configuration

このディレクトリには、Raycastの設定ファイル（エクスポートしたもの）を保存しています。

## ファイル

- `raycast.rayconfig`: Raycastの設定エクスポートファイル（パスワード保護）

## インポート方法

1. Raycastを起動
2. `Settings` (Cmd+,) を開く
3. `Advanced` タブを選択
4. `Import Configuration` をクリック
5. `raycast.rayconfig` ファイルを選択
6. パスワードを入力してインポート

## エクスポート方法（更新時）

1. Raycastを起動
2. `Settings` (Cmd+,) を開く
3. `Advanced` タブを選択
4. `Export Configuration` をクリック
5. パスワードを設定してエクスポート
6. エクスポートしたファイルを `raycast.rayconfig` にリネームして、このディレクトリに配置

## 注意事項

- 設定ファイルはパスワードで保護されているため、本人のみインポート可能
- 定期的に設定を更新した場合は、再エクスポートしてこのディレクトリに配置すること
